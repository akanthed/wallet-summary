# **App Name**: Wallet Story Explorer

## Core Features:

- Wallet Address Input: Text input for Ethereum wallet address with validation and example wallet.
- Etherscan Data Fetching: Fetch wallet data (transactions, balance, etc.) from the Etherscan API.
- AI Story Generation: Generate a wallet personality and narrative story using the Gemini API, reasoning about the Etherscan data to decide whether or not to include specifics in the generated story tool.
- Wallet Story Display: Display the AI-generated wallet personality, story, and key stats.
- Transaction History: Retrieve the transaction history of a wallet to understand user activity.

## Style Guidelines:

- Primary color: Vibrant purple (#9F55FF) to evoke curiosity and innovation in the crypto space.
- Background color: Light lavender (#F1E5FF), a very desaturated shade of the primary, to provide a gentle backdrop.
- Accent color: Electric blue (#3069FF), an analogous hue to the left of the primary, intended to pop against the more muted background.
- Headline font: 'Space Grotesk' sans-serif, for headlines and shorter text passages.
- Body font: 'Inter' sans-serif, for longer text passages.
- Use simple, clean icons to represent wallet stats and actions.
- Clean, centered layout with clear sections for input, loading, and results.
- Subtle loading animations and transitions to enhance user experience.
